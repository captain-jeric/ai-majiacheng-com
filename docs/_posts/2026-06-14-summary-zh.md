## [Google Cloud 推出 OKF：将企业知识标准化为 Markdown 供 AI Agents 使用](https://the-decoder.com/google-clouds-open-knowledge-format-turns-scattered-docs-into-markdown-files-for-ai-agents/) ⭐️ 8.0/10

Google Cloud 发布了新的开放知识格式（Open Knowledge Format, OKF），旨在将分散的企业知识标准化为带有 YAML frontmatter 的 Markdown 文件。 这使得企业知识更易于移植和被 AI Agents 利用，直接解决了 AI 应用开发中知识管理的关键挑战，对 AI 基础设施和数据准备具有潜在影响。 OKF 是一种极简规范，将组织知识结构化为 Markdown 文件，其理念与 Andrej Karpathy 提出的“LLM Wiki”概念相似。

rss · The Decoder · 6月14日 13:29

**背景**: Andrej Karpathy 的“LLM Wiki”概念旨在通过标准化和结构化的知识库，让大型语言模型（LLM）能够更高效、准确地访问和利用企业内部信息。

**标签**: `#AI Agents`, `#Knowledge Management`, `#Google Cloud`, `#Data Standardization`, `#LLM Infrastructure`
